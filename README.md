# 📍나만의 맛집 기록 서비스 (MatzipProject)

이 프로젝트는 나만의 맛집 리스트를 기록하고 관리하기 위한 웹 서비스입니다. 
🦁 멋쟁이사자처럼 14기 백엔드 과제 실습의 일환으로 제작되었습니다.

### 주요 내용

* 맛집 리스트 조회 및 등록
* 상세 리뷰 및 별점 기록
* 카테고리별 맛집 분류

---

### Tech Stack

* Python (Django)
* SQLite
* HTML / CSS (Bootstrap)
* Git / GitHub

---

### Project Structure

* **Main**: 서비스 메인 페이지 및 소개
* **Restaurants**: 맛집 목록 및 등록 기능 (ModelForm)
* **Reviews**: 방문 후기 및 평점 시스템
* **Static**: 정적 파일(CSS, JS, Images) 관리
